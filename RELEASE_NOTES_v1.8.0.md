# T-Terminal v1.8.0

## Highlights

- AI assistant session management and persistent history.
- AI chat mode and operation mode.
- Long task context compression for AI conversations.
- SFTP drag-and-drop upload fixes and chunked browser upload support.
- SFTP remote file editing through the local default editor with automatic sync.
- Host proxy tunnel for SSH targets without direct internet access.
- Terminal links now open in the system default browser.
- Terminal tab close action moved to the right-click menu to avoid accidental clicks.
- SFTP transfer speed display now resets correctly after failed or interrupted uploads.
- Old offline deployment assistant removed pending a redesign.

## Build

- Platform: Windows x64
- File: `T-Terminal-v1.8.0-windows-x64.exe`
